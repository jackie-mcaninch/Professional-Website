    Assisted design of "Libdogecoin," an open-source C library created by the Dogecoin Foundation to condense and simplify the current Dogecoin Core protocols. The 0.1 release came out in early August and contains the functions necessary for users to generate cryptographic key pairs for use on testnet or mainnet, as well as build and sign valid transactions that can be sent to the blockchain. My main contribution was writing a high-level interface for this library by creating wrappers for the library functions in both Python and Golang, along with documentation for each of these.
